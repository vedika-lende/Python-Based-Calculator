# Python-Based-Calculator 


## Problem Statement
Design and develop a Python-based calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

The application should handle user inputs efficiently, validate incorrect inputs, and display accurate results. 

The goal is to demonstrate fundamental programming concepts including conditional statements, functions, loops, and exception handling in Python. 

---

# Key Features

- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division operation
- Division by zero handling
- Menu-driven terminal interface
- Continuous execution using loops
- User-friendly output display

---

# Technologies Used

- Python
- Command Prompt / Terminal
- Jupyter Notebook

# Steps to Run the Program

## Requirements

- Python 3.x installed on system

Download Python from:

https://www.python.org/downloads/

---

# Run Instructions

## Step 1: Download or Clone the Project

```bash
git clone https://github.com/vedika-lende/Python-Based-Calculator.git
```

---

## Step 2: Open Project Folder

```bash
cd Python Based Calculator
```

---

## Step 3: Run the Program

```bash
python Calculator.py
```
